## Run Locally

Clone the project

```bash
  git clone https://github.com/nguyen-duc-loc/deadlock-tool.git
```

Go to the project directory

```bash
  cd deadlock-tool
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
